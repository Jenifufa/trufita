Descripción del Problema

Un cliente necesita un programa para manejar las reservas de un vuelo con 50 sillas:

  • 8 sillas de clase ejecutiva (filas de 4, separadas por corredor)
  • 42 sillas de clase económica (filas de 6, 3 a cada lado del corredor)

Los pasajeros proporcionan sus datos (nombre y cédula) y preferencias (clase y ubicación). Las sillas se asignan por orden de llegada según disponibilidad.

Requerimientos del Sistema

Nombre R1= RegistrarPasajero
Resumen Permite ingresar los datos personales de un pasajero en el sistema
Entradas Nombre, NumeroCedula
Resultado Los datos del pasajero son almacenados en el sistema
Nombre R2= AsignarSilla
Resumen El sistema asigna una silla disponible al pasajero según sus preferencias y la disponibilidad
Entradas NumeroCedula, ClaseDeseada, UbicacionPreferida
Resultado Una silla es asignada al pasajero por el sistema según el mapa del avión
Nombre R3= VerificarEstadoSilla
Resumen El sistema permite consultar el estado de las sillas según criterios específicos
Entradas Filtros opcionales (clase, ubicación, estado)
Resultado Las sillas disponibles se muestran con un color que se diferencia de las ocupadas
Nombre R4= CancelarReserva
Resumen EL sistema permite eliminar una reserva de silla realizada previamente
Entradas numeroCedula, numerosilla
Resultado La silla previamente reservada es cancelada en el sistema

Ejemplos de Uso

Ejemplo 1: Registrar Pasajero

El usuario ingresa los datos: Nombre = "Ana López", Cédula = "123456789". El sistema almacena esta información para futuras asignaciones.

Ejemplo 2: Asignar Silla

El pasajero con cédula "123456789" solicita clase ejecutiva, ubicación ventana. El sistema busca la primera silla disponible que cumpla estos requisitos (por ejemplo, silla 1) y la asigna.

Ejemplo 3: Ver Estado

El usuario consulta todas las sillas de clase económica en ubicación pasillo. El sistema muestra las disponibles (verde) y ocupadas (rojo).

Código de las Clases

Clase Pasajero

class Pasajero:
    def __init__(self, pCedula, pNombre):
        self.__cedula = pCedula
        self.__nombre = pNombre

    def darCedula(self):
        return self.__cedula

    def darNombre(self):
        return self.__nombre

Clase Silla

class Silla:
    class Clase:
        EJECUTIVA = "Ejecutiva"
        ECONOMICA = "Economica"

    class Ubicacion:
        VENTANA = "Ventana"
        CENTRO = "Centro"
        PASILLO = "Pasillo"

    def __init__(self, pNumero, pClase, pUbicacion):
        self.__numero = pNumero
        self.__clase = pClase
        self.__ubicacion = pUbicacion
        self.__pasajero = None

    def asignarPasajero(self, pPasajero):
        self.__pasajero = pPasajero

    def desasignarSilla(self):
        self.__pasajero = None

    def sillaAsignada(self):
        return self.__pasajero is not None

    def darNumero(self):
        return self.__numero

    def darClase(self):
        return self.__clase

    def darUbicacion(self):
        return self.__ubicacion

    def darPasajero(self):
        return self.__pasajero
